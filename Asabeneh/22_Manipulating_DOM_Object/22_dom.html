Manipulating DOM Object

Creating an Element

Creating an HTML element is simple, creating an HTML element using JavaScript is very straight forward. To create a HTML element we use document.createElement('tagName').It takes tag name as a string parameter.

Syntax:

document.createElement('tagName')

Creating Multiple Elements

To create multiple elements we use loop, after element create we can assign values to different properties of the HTML object.

Appending child to parent element

To see the created element on the HTML document we should append it to the parent element. To append the element to HTML body we use document.body which supports append.child() method.

Removing a child from a parent node easy method

To remove a element or elements from a parent element we could use removeChild() method, but let's remove it in easy way with innerHTML is equal to null.
