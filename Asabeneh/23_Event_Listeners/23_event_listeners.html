DOM : Event Listeners

Event Listeners

Common HTML events are onClick, onChange, onmouseover, onmouseout, onkeydown, onkeyup, onload. We can add event listeners to any DOM elements with the help of addEventListener() method to listen different types of events. addEventListener() method takes two arguments event listeners and callback function.

To attach the even listeners we first select the element and then add the event listener method with event type and callback function.

Syntax

selectedElement.addEventListener('onClick', (e) => { // code })
